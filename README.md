# Online Retail Store

An Amazon-inspired full-stack e-commerce web application built using the MERN stack. The deployed frontend is available at the Render link for the app titled **Online Store**. :contentReference[oaicite:0]{index=0}

## Live Demo
Frontend: https://amazon-frontend-tegq.onrender.com

## Project Overview
This project is an online retail store where users can browse products, search for items, view product details, add products to the cart, proceed to checkout, make payment, and view their orders. It is designed to provide a simple and user-friendly shopping experience.

## Features
- User registration and login
- Product listing page
- Product detail page
- Search functionality
- Add to cart
- Buy now option
- Checkout page
- Payment page
- Orders page
- Protected routes for logged-in users
- Responsive user interface
